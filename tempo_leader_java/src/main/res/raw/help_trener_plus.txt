<p><b>1 Назначение приложения.</b></p>

<p>Программа предназначена для точного дозирования циклической нагрузки за счёт задания темпа движений в процессе её выполнения. Особенно удобно применение темполидера в упражнениях с выраженными моментами силовых напряжений – подтягиваниях, отжиманиях, приседаниях и т.п. </p>

<p>Темполидер позволяет запоминать время выполнения каждого цикла движений спортсмена и затем воспроизводить его с помощью звуковых сигналов, что позволяет добиться идентичности нагрузки в двух и более подходах. Кроме того, предусмотрена возможность оперативного изменения темпа (как увеличение, так и снижение) на любую заданную в процентах величину.</p>

<p>Если темп в ходе выполнения упражнения изменяется скачкообразно – как это часто бывает в подтягивании на перекладине, когда приходится согласовывать циклы подтягиваний с циклами дыхания – время выполнения цикла движений может быть одинаковым для нескольких циклов подряд. В этом случае подход фактически распадается на фрагменты с различным темпом движений, что позволяет быстро и удобно вручную составлять раскладки по темпу, задавая время выполнения цикла и количество повторений для всех фрагментов подхода</p>

<p>Темполидер, позволяя воспроизводить раскладку по темпу любой степени сложности, даёт возможность контролировать темп выполнения упражнения от подхода к подходу, от тренировки к тренировке, помогая объективно оценивать изменение уровня работоспособности спортсмена.</p>

<br>
<p><b>2 Экран «Главное меню». Выбор варианта действий.</b></p>

<p>Пункт <b>«Секундомер»</b> открывает экран <b>«Секундомер»</b> с отсечками и возможностью сохранения отсечек в виде раскладки секундомера, где количество повторений каждого фрагмента равно 1.</p>

<p>Сохранённые отсечки можно просмотреть на графике и визуально оценить степень изменения темпа в процессе выполнения упражнения.</p>

<p>Пункт <b>«Темполидер»</b> открывает экран <b>«Темполидер»</b> который помогает выполнить циклическое упражнение в соответствии с заданной раскладкой по темпу со звуковым сопровождением.</p>

<p>Пункт <b>«Раскладки»</b> открывает экран «Раскладки» с тремя вкладками, содержащими списки раскладок, сохранённых в базе данных приложения.</p>

<br>
<p> <b>2.1 Создание новой раскладки по темпу.</b></p>

<p> <b>2.1.1 Через круглую кнопку «+» в главном меню</b></p>

<p>Введите данные хотя бы для одного фрагмента подхода, укажите время задержки старта, дайте раскладке имя, а затем нажмите <b>«Создать раскладку»</b></p>

<p> <b>2.1.2 На основе ранее созданной раскладки</b></p>

<p>Для этого выберите пункт <b>«Раскладки»</b> в главном меню, а затем выберите подходящую раскладку на одной из трёх вкладок. Выбранная раскладка откроется на экране темполидера. Для редактирования раскладки перейдите в редактор раскладок, нажав на иконку карандаша на панели инструментов в верхней части экрана.</p>

<p> Доступно изменение параметров для всех фрагментов сразу или для любого выбранного фрагмента в зависимости от установки пункта <b>«Выполнить для всех строк»</b>, Добавление, удаление или вставка фрагмента доступны из <b>меню</b>, которое появляется при долгом нажатии на строке любого фрагмента</p>

<p> <b>2.1.3 Путём фиксации длительности циклов движений с помощью секундомера с отсечками</b></p>

<p>Для некоторых упражнений (например, приседаний) это можно сделать самостоятельно, для большинства других придётся прибегнуть к посторонней помощи.</p>

<p>Кроме того, можно записать раскладку своего друга, соперника и т.д. и попробовать повторить или превзойти их результат!</p>

<br>
<p><b>3 Экран «Секундомер». Использование секундомера.</b></p>

<p><b>3.1 Использование секундомера для создания раскладки по темпу</b></p>

<p>Секундомер с отсечками позволяет делать отсечки времени в ходе выполнения какого либо упражнения. Если делать отсечки в каждом цикле циклического упражнения, можно получить достаточно подробную и информативную раскладку по темпу, каждый фрагмент которой включает время между соседними циклами движений и количество повторений равное 1.</p>

<p>После нажатия на кнопку <b>«Стоп»</b> появляется диалог сохранения раскладки секундомера, в котором задаётся имя раскладки и тип сохранения.</p>

<p>При выборе пункта <b>«Сохранить и показать»</b> вызывается экран <b>«График»</b>, а отсечки отображаются на графике, что позволяет визуально оценить степень изменения темпа в процессе выполнения упражнения.</p>

<p>При нажатии на стрелку справа в панели инструментов раскладка загружается в темполидер, где может быть воспроизведена или отредактирована</p>

<p><b>3.2 Использование секундомера для записи интервалов времени.</b></p>

<p>Кроме создания раскладок секундомер может использоваться и в обычном режиме записи отсечек по кругам, например при беге.</p>

<br>
<p><b>4 Экран «Темполидер». Использование темполидера.</b> </p>

<p>При выборе пункта Главного меню <b>«Темполидер»</b> вы попадаете в основной экран приложения, который позволяет выполнять циклическое упражнение в соответствии с заданной раскладкой по темпу.</p>

<p>Имя выбранной раскладки показано в верхней части экрана.</p>

<p><b>Задержку старта</b> для подготовки к выполнению упражнения можно задать при нажатии на кнопку <b>«Задержка, сек»</b>.</p>

<p>Если в <b>Настройках</b> включен звук, выполнение упражнения сопровождается звуковыми сигналами, в противном случае можно ориентироваться на цветовые подсказки.</p>

<p>После окончания упражнения включается <b>таймер отдыха</b>, повторное нажатие на кнопку <b>«Старт»</b> запускает следующий подход с повторной задержкой старта.</p>

<p>Список раскладок, режим редактирования раскладок и настройки доступны при нажатии на иконки в панели инструментов, кроме того в режим редактирования можно перейти при долгом нажатии на любом фрагменте подхода.</p>

<p><b>4.1 Экран «Изменить». Редактирование раскладок.</b></p>

<p>Доступно изменение параметров (время цикла или количество повторений) для всех фрагментов сразу или для любого выбранного фрагмента в зависимости от установки пункта <b>«Выполнить для всех строк».</b></p>

<p>Изменение, удаление или вставка фрагмента доступны из <b>меню</b>, которое появляется при долгом нажатии на строке любого фрагмента.</p>

<p>Добавление фрагмента подхода в конец списка фрагментов, а также изменение темпа выполнения упражнения в целом доступно из панели инструментов</p>

<p>При каком либо изменении параметров нагрузки на панели инструментов появляется иконка <b>«Сохранить как»</b> , взывающая соответствующий диалог сохранения раскладки под новым именем.</p>

<br>
<p><b>5 Экран «Раскладки». Списки раскладок.</b> </p>

<p>Экран включает в себя три вкладки, содержащие списки раскладок, сохранённые в базе данных приложения.</p>

<p>Вкладка <b>«Сек»</b> изначально содержит раскладки по темпу, созданные из <b>«Секундомера»</b></p>

<p>Вкладка <b>«Темп»</b> изначально содержит раскладки по темпу, созданные или отредактированные и сохранённые из <b>«Темполидера»</b></p>

<p>Вкладка <b>«Избр»</b> изначально содержит раскладки по темпу, созданные из главного меню при нажатии на круглую кнопку <b>«+»</b>.</p>

<p>Перемещение по вкладкам, удаление или изменение имён раскладок доступно из выпадающего меню при долгом нажатии на строку с названием раскладки.</p>

<p>Исключение составляют системные раскладки <b>«автосохранение_секундомера»</b> и <b>«автосохранение_темполидера»</b>, манипуляции с которыми запрещены на программном уровне.<br>
</p>
